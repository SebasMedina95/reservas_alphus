Si en vez de trabajar con "localhost" como nombre de URL queremos trabajar con un "dominio" que
en ultimas termina siendo es un dominio virtual, debemos hacer una serie de configuraciones para
conseguir el resultado, no es muy complejo, PERO, debemos tener muy presente la dirección donde
esta alojado nuestro XAMPP y donde estará ubicado nuestro archivo host del equipo para aplicar
las configuraciones para la dirección 127.0.0.1.

Entonces:

1. Configuración del archivo host.
   *** Nos dirigimos a: C:\Windows\System32\drivers\etc y el archivo host lo editamos:
       Lo que vamos a editar son las resoluciones DNS, quedando:
	
# localhost name resolution is handled within DNS itself.
	127.0.0.1       localhost
	127.0.0.1       api.marketplace-ladum.com
	127.0.0.1       marketplace-ladum.com
	127.0.0.1       reservas-alphus.com
#	::1             localhost

Como podemos aprediar, tenemos el localhost default que lo debemos conservar para acceder a las configuraciones
generales del XAMPP en temas del Apache, el PhpMyAdmin, entre otras configuraciones importantes. Colocamos la IP
que en este caso es la local 127.0.0.1 al lado colocamos "el dominio".

2. Para poder reconocer a que proyecto se refiere, debemos crear un host virtual, esto lo hacemos en la carpeta
de configuración del XAMPP, vamos a buscar el archivo httpd-vhosts.conf dentro del XAMPP, que se encuentra en
la ruta de la versión 8: E:\Desarrollo\xampp-v8\apache\conf\extra  (recordemos que la ruta varia conforme
al lugar donde se hizo la instalación del XAMPP). Luego allí, generamos los virtual host al final, quedando:

##########################################################################################

<VirtualHost *:80>
    DocumentRoot "E:/Desarrollo/xampp-v8/htdocs/"
    ServerName localhost
</VirtualHost>

<VirtualHost *:80>
    DocumentRoot "E:/Desarrollo/xampp-v8/htdocs/api-marketplace-ladum"
    ServerName api.marketplace-ladum.com
</VirtualHost>

<VirtualHost *:80>
    DocumentRoot "E:/Desarrollo/xampp-v8/htdocs/marketplace-ladum"
    ServerName marketplace-ladum.com
</VirtualHost>

<VirtualHost *:80>
    DocumentRoot "E:/Desarrollo/xampp-v8/htdocs/reservas-alphus"
    ServerName reservas-alphus.com
</VirtualHost>

##########################################################################################

Es muy pero muy importante tener en cuenta el puerto que tenemos configurado para el tema, por lo que, en este
documento tenemos el puerto 80, pero, podría variar a 50, o 8080, depende de la configuración que le tengamos,
como vemos, en el DocumentRoot debemos apuntar exactamente a la carpeta del proyecto, y, el ServerName tendrá
el dominio que deseamos y que esta configurado previamente en mi archivo host.

3. Debemos reiniciar el XAMPP y ya deberíamos acceder no con localhost si no con el dominio virtual creado.

NOTA: Si en el proyecto configuraste aspectos con "localhost" debes ir a cambiar esto para que te tome los 
cambios también, si no, no te lo detectará y tendrás errores en la aplicación
